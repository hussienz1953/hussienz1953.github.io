## ğŸ”§ Features

OpenWorkers Infrastructure offers multiple components for a robust environment:

- **PostgreSQL Database:** Reliable storage for your data.
- **NATS Message Queue:** Effective communication between workers.
- **Postgate:** An HTTP proxy that validates database queries.
- **OpenWorkers API:** A RESTful API to interact with your workers.
- **OpenWorkers Runner:** Executes your worker scripts in a secure environment.
- **OpenWorkers Logs:** Collects and organizes log data for easy access.
- **OpenWorkers Scheduler:** Manages scheduled tasks to run at specified times.
- **OpenWorkers Dashboard:** User-friendly interface for monitoring your application.
- **OpenWorkers Proxy:** Directs web traffic to the appropriate service.
